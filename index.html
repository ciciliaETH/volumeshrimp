1. Start the Bot :
Send the command /start to initiate the bot and display the main menu

2. Create Wallet :
Select "Create Wallet" from the menu to generate a new wallet. The bot will provide you with a public key and a private key

3. Import Wallet :Select "Import Wallet" and send your private key to the bot. It will import the wallet and provide the public key

4. Show Private Key :Select "Show Private Key" to display the private keys of all wallets you have created or imported

5. Set Pair :Select "Set Pair" and send the token address you want to pair for the volume booster.

6. Start Booster :Select "Start Booster" to begin the volume boosting process. Ensure you have a wallet and token pair set, as well as purchase limits configured

7. Stop Booster :Select "Stop Booster" to halt the volume booster if it is running.

8. View Wallet :Select "View Wallets" to see a list of your wallets along with their SOL balances.

9. Set Buy Limits :Select "Set Buy Limits" and send the minimum and maximum purchase limits in the format: minAmount maxAmount (e.g., 0.0001 0.001)

10. Back To Main Menu :Select "Back to Main Menu" to return to the main menu from any submenu
